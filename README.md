## **Objetivo**
Fazer um jogo-texto **[Mud](https://www.theenemy.com.br/games/mud-a-moda-mais-antiga-de-multiplayers-da-rede)** aplicando os conteudos vistos em sala de aula sobre banco de dados relacionais usando postgres e python. 


## **Requisitos**

Para instalar os pacotes a seguir temos como base o gerenciador de pacotes `pip` que vem da liguagem [python](https://www.python.org/downloads/).

O banco de dados usado no projeto é o [postgres](https://www.postgresql.org/) no qual usamos a liguagem [SQL](https://www.alura.com.br/artigos/o-que-e-sql) e adjunto com o python criamos a lógica e o banco do nosso jogo

1. Aqui temos o pacote usado para fazer as mediações do banco de dados postgres com a liguagem python

    ```bash
    pip install psycopg2-binary 
    Versão = psycopg2-binary==2.9.6
    ```
